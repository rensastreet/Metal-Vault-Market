## Packages
react-simple-maps | Interactive world map visualization for vault locations
d3-geo | Required for custom map projections and math
framer-motion | Premium animations and transitions for the UI
recharts | Portfolio data visualization and charts
date-fns | Formatting transaction timestamps
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
- Using 'Outfit' for display fonts and 'Inter' for body text to create a premium fintech aesthetic.
- The map uses a public TopoJSON file from a CDN for the base world geometry.
- Assumes `use-auth.ts` is available as per the existing frontend files context.
- Colors are strictly dark mode to fit the premium precious metals trading vibe.
